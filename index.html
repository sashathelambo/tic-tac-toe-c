Instructions

This is a console-based Tic-Tac-Toe game written in C++. Follow the instructions below to play the game:

  1. Download and extract the game files.
  2. Open a terminal or command prompt.
  3. Navigate to the directory containing the game files.
  4. Run the executable file main.exe to start the game.
  5. Follow the on-screen instructions to play against another player or the computer.
  6. Press ESC at any time to exit the game or M to return to the menu.

Game Features

  • Play against another player or the computer.
  • Color-coded console output for better visibility.
  • Simple and intuitive controls.
